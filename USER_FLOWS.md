# EduConnect User Flows

This document summarizes the user flows and navigation paths for each role in the EduConnect application.

## üéì Student Flow

### 1. Dashboard (Home)
The central hub for student activities.
- **View Announcements:** Read latest university news and updates.
- **Check Progress:** View progress bars for enrolled courses.
- **Monitor Deadlines:** See upcoming assignments and exams in the "Upcoming Deadlines" sidebar.
- **Check Notifications:** View alerts for schedule changes or assignment reminders.
- **Interactive Calendar:**
  - **Action:** Click on a date (e.g., May 13).
  - **Result:** A modal opens showing the class schedule (Time, Subject, Room, Teacher) for that day.

### 2. My Courses & Learning
- **Navigation:** Click "My Courses" in sidebar.
- **View Courses:** See a list of all enrolled courses.
- **Access Course Details:**
  - **Action:** Click **"View Course"** on a course card.
  - **Result:** Navigate to **Course Details** page.

#### Course Details Page
- **Materials:** View and download lectures/labs.
- **Assignments:**
  - **Action:** Click **"Submit"** on an assignment.
  - **Result:** Navigate to **Assignment Details** page to upload files.
  - **Action:** Click **"View Submission"** (if already submitted) to see status.
- **Progress Tests (Quizzes):**
  - **Action:** Click **"Start Test"** on an available quiz.
  - **Result:** Enter the **Quiz Interface**.

#### Quiz Interface
- **Take Exam:** Answer multiple-choice questions with a countdown timer.
- **Navigation:** Use the Question Navigator to jump between questions.
- **Tools:** Flag questions for review.
- **Submission:** Click "Submit Exam" to finish.

### 3. Profile & Grades
- **Navigation:** (Via UI/Header)
- **Student Profile:** View GPA, academic standing, and full course history.

### 1. Dashboard
Overview of teaching responsibilities.
- **Stats:** View total students, classes, and hours taught.
- **Upcoming Classes:** See the immediate schedule.
- **Recent Activity:** Track student submissions and system alerts.

### 2. Class Management
- **Navigation:** Click "Classes" in sidebar.
- **View Classes:** List of classes currently being taught.
- **Manage Class:**
  - **Action:** Click **"Manage Class"** on a class card.
  - **Result:** Navigate to **Class Details** page.

#### Class Details Page
- **Student Roster:** View list of students with attendance and current grades.
- **Assignments:** Monitor submission counts and average grades.
- **Quick Actions:** Buttons to "Take Attendance" or "Grade Assignments".

### 3. Grading & Analytics
- **Grading:** (Placeholder) Interface to input grades.
- **Analytics:** (Placeholder) Charts showing class performance trends.

### 1. Dashboard
System-wide overview.
- **System Stats:** Total Users, Active Courses, Revenue, System Health.
- **Revenue Chart:** Visual breakdown of income.
- **Quick Actions:** Add User, Add Course, Generate Reports.

- **Navigation:** Click "Users" in sidebar.
- **Action:** View, add, edit, or delete student/teacher accounts.
